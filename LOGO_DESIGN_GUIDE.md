### 方法 2：使用在线工具（推荐，最专业）

#### 🌟 Canva（免费，功能强大）

**步骤**：

1. **访问 Canva**
   - 网址：https://www.canva.com/
   - 注册/登录（可用 Google 账号）

2. **选择模板**
   - 搜索 "Logo"
   - 筛选：Technology / Security / Software
   - 选择简洁的模板

3. **自定义设计**
   
   **文字部分**：
   - 主标题：`IoT Defense` 或 `Replay Shield`
   - 副标题：`Security Simulator` 或留空
   - 字体推荐：
     - Montserrat（现代、专业）
     - Roboto（清晰、技术感）
     - Inter（简洁、易读）

   **图标部分**（从 Canva 图标库选择）：
   - 🛡️ 盾牌（安全）
   - 🔒 锁（防护）
   - 🌐 网络节点（IoT）
   - 🔄 循环箭头（重放攻击）
   - 💾 芯片（技术）

   **颜色方案**：
   ```
   主色：#2563EB（蓝色 - 信任、安全）
   辅色：#F59E0B（橙色 - 警告、重要）
   强调：#10B981（绿色 - 成功、验证）
   文字：#1F2937（深灰 - 专业）
   ```

4. **导出设置**
   - 文件类型：PNG
   - 背景：透明
   - 尺寸：
     - 图标：512×512 px（GitHub 头像）
     - 横幅：1200×400 px（README 顶部）
     - 社交媒体：1200×630 px（分享卡片）

## 🎨 设计元素建议

### 核心概念

你的项目是关于 **IoT 设备的重放攻击防御**，Logo 应该传达：
1. **安全性**（盾牌、锁）
2. **技术性**（网络、芯片）
3. **防御性**（保护、阻挡）

### 推荐图标组合

#### 组合 1：盾牌 + 锁（经典）
```
    🛡️
    🔒
```
**寓意**：安全防护、可靠保障

#### 组合 2：网络 + 盾牌（IoT 特色）
```
  🌐
  🛡️
```
**寓意**：物联网安全、网络防护

#### 组合 3：芯片 + 锁（技术感）
```
  💾
  🔐
```
**寓意**：硬件安全、技术防护

#### 组合 4：循环箭头 + 禁止符号（直观）
```
  🔄
  🚫
```
**寓意**：阻止重放攻击

## 💡 设计技巧

### 1. 保持简洁
- ❌ 避免：过多细节、复杂渐变
- ✅ 推荐：简单形状、纯色填充

### 2. 可识别性
- ❌ 避免：抽象难懂的图形
- ✅ 推荐：清晰的盾牌/锁图标

### 3. 可缩放性
- ❌ 避免：细线条、小文字
- ✅ 推荐：粗线条、清晰轮廓

### 4. 颜色使用
- ❌ 避免：超过 3 种主色
- ✅ 推荐：2-3 种协调色

## ❓ 常见问题

### Q1: 我不会设计，怎么办？
**A**: 使用 Canva 的模板！选择一个科技类模板，只需修改颜色和文字即可。

### Q2: Logo 需要注册商标吗？
**A**: 对于开源项目，通常不需要。MIT License 已经足够。

### Q3: 应该用 PNG 还是 SVG？
**A**: 两者都准备：
- PNG：用于 GitHub、社交媒体（易用）
- SVG：用于网站、印刷（可缩放）

### Q4: Logo 可以后期修改吗？
**A**: 可以！开源项目的 Logo 可以随时更新。
